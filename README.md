# language-school-management-system
Role-based language school management system built with ASP.NET Core Web API, featuring controlled course approval workflows and AI-assisted message classification.

## Tech Stack
- ASP.NET Core Web API
- EF Core + SQLite
- HTML/CSS/JS frontend
- Python FastAPI AI service

## Core Features
- Role-based access (Visitor / Student / Admin)
- Course request workflow with approval
- AI-assisted message classification
